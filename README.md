# 康立数字软件服务平台

## 项目简介
康立数字软件服务平台是一个专业的微信服务号展示系统，用于展示康立团队的产品和解决方案。

## 功能模块
- **关于康立**: 康立数字简介、官网、云平台入口、联系我们
- **产品系列**: PLC系统、数采网关、标品MES、设备TPM、智能工具柜
- **解决方案**: 定制化控制、设备上云、家纺服装、注塑机方案

## 技术栈
- Vue 3
- Vue Router
- Vite
- 响应式设计

## 运行项目
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 项目特点
- 现代化UI设计
- 底部导航栏切换
- 支持文字、图片、视频展示
- 移动端优化
- 响应式布局
